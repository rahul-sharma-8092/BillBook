# BillBook Frontend

React + TypeScript + Tailwind UI for local cash memo billing.

## Run (Dev)

1. `npm install`
2. `npm run dev`

The Vite dev server proxies `/api` to `http://localhost:5064` (see `vite.config.ts`).

## Build

1. `npm run build`
2. Deploy `dist/` as a static site (IIS, nginx, etc.)
